Azure IoT Edge Pipeline

Azure IoT Edge Pipeline

Overview

This project demonstrates a complete Azure IoT Edge data pipeline that showcases modern edge computing architecture for IoT telemetry collection and processing. The system implements a realistic edge-to-cloud data flow using containerized IoT Edge modules running on Ubuntu Core, with seamless integration to Azure cloud services.

The architecture simulates a temperature sensor deployed as a Docker container on an IoT Edge device, transmitting data through Azure IoT Hub to Event Hubs for real-time processing and analysis.

Key Features

  • Containerized Edge Modules: Docker-based IoT Edge modules
  • Ubuntu Core Integration: Snap-based Azure IoT Edge runtime on Ubuntu Core
  • Real-time Telemetry: Simulated temperature sensor with continuous data streaming
  • Azure Cloud Integration: Seamless connection to Azure IoT Hub and Event Hubs
  • Container Registry: Azure Container Registry (ACR) for module image management
  • Event-Driven Architecture: Event Hub consumer for real-time data processing
  • Reproducible Setup: Complete step-by-step guide for replicating the entire pipeline for free

Technology Stack

  • Edge Runtime: Azure IoT Edge on Ubuntu Core
  • Containerization: Docker containers for IoT modules
  • Cloud Services: Azure IoT Hub, Azure Container Registry
  • Programming: Python 3.8+ for IoT modules and event consumers
  • Authentication: Symmetric key authentication for IoT Edge devices
  • Infrastructure: Virtual machine deployment with Ubuntu Core

Architecture Components

Edge Layer

  • IoT Edge Device: Ubuntu Core VM running Azure IoT Edge runtime
  • Temperature Sensor Module: Containerized Python application simulating sensor data

Cloud Layer

  • Azure IoT Hub: Central message hub for bi-directional communication
  • Azure Container Registry: Private container image repository
  • Event Consumer: Python application for processing telemetry data

Repository

The repository includes all source code and detailed documentation for reproducing the entire Azure IoT Edge pipeline from scratch.
